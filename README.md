# To-Do List Application

## Project Overview

The To-Do List Application is a simple task management system developed using Python and Tkinter. It helps users organize their daily activities by allowing them to add, delete, and mark tasks as completed through an easy-to-use graphical interface.

## Features

* Add new tasks
* Delete existing tasks
* Mark tasks as completed
* User-friendly GUI
* Easy task management

## Technologies Used

* Python
* Tkinter

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Run the following command:

```bash
python todo.py
```

4. The To-Do List application window will open.
